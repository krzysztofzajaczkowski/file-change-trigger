GitHub Action that triggers specified repository's workflow if selected file or directory was updated. 
Check is based on latest commit that changed directory/file and latest successful workflow run. 
If updates occurred later than latest completed workflow - the workflow is triggered. 
This Action is used for triggering <Underlining highlight><a href="https://github.com/krzysztofzajaczkowski/distributed-portfolio" target="_blank" rel="nofollow noopener noreferrer" aria-label="External Link">distributed portfolio</a></Underlining> deployment workflow when any project with this job set up has portfolio description updated.
